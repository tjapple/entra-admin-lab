# Role Assignment

## Purpose
Assign or adjust administrative roles for a user directly from the user’s profile.

---

## Procedure

### 1. Navigate to the User
- Azure Portal → Microsoft Entra ID
- Go to **Users → All users → select the user**

### 2. Open Assigned Roles
In the left menu:
- Select **Assigned roles**

### 3. Add or Remove Roles
- Click **Add assignments** tab
- Assign only the minimum required role
- Select any role assignments you wish to remove
- Click **Remove assignments** tab
  
### 4. Verify Changes
- Confirm the role now appears under **Assigned roles**
- Confirm unwanted roles were removed

---

## Notes
- Some tenants no longer allow role assignment from “Roles and administrators”; everything is done per-user.
- Role-assignable groups may not appear unless explicitly created.
